# 🧠 FocusMirror — Real-Time Distraction & Focus Tracker

## 🚀 Overview

FocusMirror is a productivity web application built using React that helps users understand their focus patterns during study or work sessions.

Instead of just tracking tasks, FocusMirror analyzes **how well you stay focused** and provides insights into your distractions, helping you improve productivity over time.

---

## 💡 Problem It Solves

Most productivity tools focus on *what* you plan to do.  
FocusMirror focuses on *how* you actually perform.

It answers questions like:

* How often do I get distracted?
* When is my focus at its peak?
* How consistent is my study routine?

---

## ⚙️ Key Features

### ⏱️ Focus Timer

Start a focused work session using a timer (similar to Pomodoro).  
Users can define session duration and track uninterrupted work time.

---

### 🚨 Distraction Tracker

With a single click, users can log moments when they lose focus.  
Each distraction is recorded during the session for analysis.

---

### 📊 Focus Score

At the end of each session, FocusMirror calculates a **Focus Score** based on time spent focusing vs. distractions.

This helps quantify productivity in a simple, understandable way.

---

### 📈 Analytics Dashboard

Users can view:

* Total focus time
* Number of distractions
* Daily/weekly performance trends
* Session history

---

### 🧠 Smart Insights

FocusMirror generates simple behavioral insights such as:

* Peak productivity hours
* Average distraction frequency
* Suggestions to improve focus

---

### 💾 Local Storage Persistence

All data is stored in the browser using localStorage, allowing users to retain their session history without needing a backend.

---

## 🛠️ Tech Stack

* React (Frontend)
* HTML
* CSS
* JavaScript
* Browser LocalStorage

---

## 🧩 How It Works

1. User starts a focus session
2. Timer runs in real-time
3. User logs distractions manually
4. Session data is saved locally
5. Focus score and insights are generated
6. Data is visualized on the dashboard

---

## 🎯 Use Cases

* Students preparing for exams
* Developers working in deep focus sessions
* Anyone looking to improve concentration and productivity

---

## 🏁 Conclusion

FocusMirror is not just a tracker—it’s a **self-awareness tool** that helps users understand and improve their focus habits through simple interaction and meaningful insights.
